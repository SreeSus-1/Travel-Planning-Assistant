# Travel-Planning-Assistant
Help users collaboratively plan trips with AI agents.


This project is a multi-agent travel planning system that helps users create personalized trip plans based on destination, budget, and interests. It uses an Itinerary Builder Agent to create a day-by-day travel plan, a Cost Estimator Agent to calculate likely expenses for stay, transport, food, and activities, and a Local Culture Coach Agent to provide practical advice about etiquette, cultural norms, food, safety, and local experiences. The agents work together to produce a complete trip plan rather than isolated outputs. User inputs and generated plans are stored in memory so past trips, travel preferences, and notes can be revisited later through the dashboard.
